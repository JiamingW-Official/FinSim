# Skill: empty-state
Add proper empty states to all pages/sections that can be empty. Rules:
- Every list/table that can be empty needs an empty state
- Empty state format: centered icon (from lucide-react) + title + description + optional CTA button
- Use muted colors (text-muted-foreground)
- Icon size: h-8 w-8 with text-muted-foreground/50
- Title: text-sm font-medium
- Description: text-xs text-muted-foreground
- CTA: small outline button pointing to relevant action
- No sad faces or overly emotional empty states
Find missing empty states, add them. Report additions.
