# Skill: layout-consistency
Ensure every page follows the same layout pattern:
- Page header: consistent height, icon + title + subtitle pattern
- Content area: consistent padding (p-4 or p-6)
- Grid layouts: use consistent column counts (1, 2, 3, or 4 columns)
- Responsive: all grids should have responsive breakpoints (grid-cols-1 sm:grid-cols-2 lg:grid-cols-3)
- No orphaned sections or floating elements
- Consistent section spacing (space-y-4 or space-y-6)
- Bottom bar (status bar) should be consistent height and content
Audit layout structure, fix inconsistencies. Report per page.
