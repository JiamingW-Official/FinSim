# Skill: breathing-rhythm
Break "over-engineered uniform spacing" feel. Rules:
- Hero sections: p-6 (generous breathing room)
- Action sections: p-4 (standard)
- Reference sections: p-3 (compact, pushed down)
- NOT all sections same padding — vary by importance
- Add whitespace BETWEEN tiers (mb-6 between hero and action, mb-4 between action and reference)
- Section headers: vary weight — tier 1 text-sm font-semibold, tier 2 text-xs font-medium, tier 3 text-xs text-muted-foreground
- Grid gaps: vary by tier — gap-4 for hero, gap-3 for action, gap-2 for reference
