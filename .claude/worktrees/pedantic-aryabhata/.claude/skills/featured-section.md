# Skill: featured-section
Add "featured" hero sections to list/catalog pages. Pattern:
- Pick highest-engagement or most-relevant item
- Display as hero card (border-l-4 border-primary p-6)
- Larger text (text-lg font-medium), prominent CTA button
- Below: educational one-liner in text-xs text-muted-foreground
- Rest of list labeled "All [Items]" with count badge
Apply to: Predictions (featured market), Learn (up next lesson), Challenges (daily challenge)
