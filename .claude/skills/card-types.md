# Skill: card-types
4 distinct card types — break visual uniformity:
- HERO: `border-l-4 border-primary bg-card p-6 rounded-lg` (1 per page, primary content)
- DATA: `bg-muted/30 p-3 rounded-md` (no border, compact stats/metrics)
- INTERACTIVE: `border border-border bg-card p-4 rounded-lg hover:border-primary/50 transition-colors cursor-pointer` (CTAs, lessons, actions)
- REFERENCE: `border-border/30 p-3 rounded-md` (subtle, secondary info, timelines)
Apply based on content type, not uniformly.
