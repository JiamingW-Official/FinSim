# Skill: responsive-fix
Fix responsive design issues. Rules:
- All pages must work at 1024px, 1280px, and 1440px widths
- Tables should have horizontal scroll on smaller screens
- Cards should stack on mobile (grid-cols-1)
- Sidebar should collapse to icons at <1024px
- No horizontal overflow on any page
- Text should not truncate in ways that lose meaning
- Modals should be max-w-lg and centered
Audit responsive breakpoints, fix issues. Report per page.
