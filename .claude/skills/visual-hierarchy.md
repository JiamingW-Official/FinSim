# Skill: visual-hierarchy
Create clear visual power structure on every page. Rules:
- TIER 1 (Hero Zone, top 60%): border-l-4 border-primary, p-6, larger text, ONE per page max
- TIER 2 (Action Zone): bg-muted/30 p-3 rounded-md, compact stats/numbers, no borders
- TIER 3 (Reference Zone): border-border/30 p-3, text-xs headers, muted, below fold
- Every page MUST have exactly 1 hero element that dominates
- Stats rows: compact, no icons, label+number only
- Quick actions: max 4, not 8
- Section headers below fold: text-xs font-medium text-muted-foreground, NO icons
