# Skill: onboarding-flow
Optimize the onboarding and "Start Here" flow. Goals:
- Onboarding should take <60 seconds
- Clear 3-step path: Choose interest → See first lesson → Place first trade
- Progress indicators visible at all times
- "What to do next" always obvious on every page
- Home dashboard should have a prominent "Continue your journey" section
- Remove any confusing or skippable steps
- After onboarding, redirect to the first lesson (not home)
Audit onboarding files, redesign flow. Report changes.
