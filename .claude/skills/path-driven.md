# Skill: path-driven
Convert catalog/list pages to path-driven UX. Pattern:
- "Up Next" hero card at top (border-l-4 border-primary p-6, large CTA)
- Progress journey: compact 3-step dots (Learn→Practice→Review)
- Active path highlighted, completed paths checkmarked
- "All Items" tab is secondary with smaller cards (p-3, grid-cols-3)
- Every completed item shows "Practice this →" link
- Bottom tip: educational reinforcement message
Apply to: Learn page, Challenges page
