# Skill: spacing-grid
Enforce consistent spacing grid across all pages. Rules:
- Page padding: p-4 or p-6 (not p-3, p-2, p-5, p-8)
- Card padding: p-4 (not p-3 or p-2.5)
- Section gaps: gap-4 or gap-6 (not gap-2, gap-3, gap-8)
- Card gaps in grids: gap-4
- Inner element gaps: gap-2 or gap-3
- Border radius: rounded-lg for cards, rounded-md for buttons/inputs, rounded-full for badges/avatars ONLY (no rounded-xl, rounded-2xl, rounded-3xl)
- Remove ALL rounded-3xl, rounded-2xl usage
- Standardize card borders: border border-border (not border-2, not colored borders)
Find violations, fix. Report changes per file.
